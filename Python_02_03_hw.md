## Задание 9.
В программу вводится число N. Программа должна вывести два столбца чисел. В первом столбце – числа от 1 до N, во втором – числа от N до 1.
```
n=int(input())
s = 0
m = n + 1
for i in range(n):
    s += 1
    m -= 1
    print(s, m, sep=' ')
```
Пример:  
5  
1 5  
2 4  
3 3  
4 2  
5 1  

## Задание 10.
В программу вводится число N. Требуется определить, является ли это число простым. 
```
import random
n=random.randint(1,1000)
print('n =', n)
if n < 2:
    print('Сложное')
else:
    is_prime = True
    for i in range(2,n):
        if n % i == 0:
            is_prime = False
            break
print('Простое' if is_prime else 'Сложное')
```
